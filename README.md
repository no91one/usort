# Usort

---

> It's a multifunctional npm package for using various data structures and algorithms in your project.

### Installation

```bash
npm i usort
```

## Features

    * Sorting
        * Bubble Sort
        * Selection Sort
        * Insertion Sort
        * Merge Sort

    * Maths
        * GCD
        * LCM
        * Square root
        * Cube root
        * Exponent

    * Searching Algorithms
        * Binary Search
        * Linear Search

    * Graph Algorithms
        * DFS
        * BFS
